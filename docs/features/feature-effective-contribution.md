# Especificação Funcional - Feature: Effective Contribution

## Objetivo

Definir a experiência do usuário ao compreender a Contribuição Efetiva de uma
Sessão de desenvolvimento analisada pelo DevIQ.

## Problema

O desenvolvedor deseja entender o valor realmente produzido durante a sessão,
não apenas quanto código foi escrito ou quantos prompts foram enviados.

## Perguntas Respondidas

- Qual foi a contribuição efetiva da sessão?
- Como a IA influenciou o resultado?
- Onde ocorreu maior participação humana?
- Quais entregas geraram maior valor?
- Quais atividades tiveram baixo retorno?

## Conteúdo Obrigatório

- Contribuição geral da sessão
- Distribuição IA × Desenvolvedor
- Principais entregas
- Evidências de suporte
- Indicadores relevantes
- Fatores que aumentaram ou reduziram a contribuição

## Navegação

Cada indicador deve permitir aprofundamento até:

- Métrica
- Insight
- Evidência
- Prompt relacionado

## Critérios de Aceitação

- A contribuição é apresentada em linguagem compreensível.
- Toda conclusão é rastreável.
- O usuário consegue identificar rapidamente onde agregou mais valor durante a sessão.
