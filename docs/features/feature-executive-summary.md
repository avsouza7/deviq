# Especificação Funcional - Feature: Executive Summary

## Objetivo

Definir a experiência do usuário ao consumir o Executive Summary produzido pelo
DevIQ após a execução do `deviq run`.

## Problema

Após uma longa sessão de desenvolvimento assistido por IA, o desenvolvedor não
deve precisar revisar centenas de prompts, commits e arquivos para entender o
que realmente aconteceu.

## Resultado Esperado

O Executive Summary deve responder rapidamente:

- O que foi entregue?
- O que mudou?
- Quais decisões foram tomadas?
- O que foi aprendido?
- Qual foi a contribuição efetiva?
- O que merece atenção no próximo ciclo?

## Conteúdo Obrigatório

- Resumo executivo
- Effective Contribution
- Principais métricas
- Decisões arquiteturais
- Aprendizados
- Dívida técnica
- Recomendações
- Próximos passos

## Navegação

Cada seção deve permitir aprofundamento até:

- Insight
- Knowledge
- Evidence

## Critérios de Aceitação

- Leitura em poucos minutos.
- Todas as afirmações são rastreáveis.
- O usuário compreende o resultado da sessão sem abrir o relatório completo.
