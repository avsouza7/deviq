# Execution Decision Model

## Objetivo

Definir como o DevIQ decide quais artefatos gerar durante uma execução.

## Fluxo

1. Compreender a Sessão.
2. Identificar intenção.
3. Avaliar impacto.
4. Selecionar artefatos.
5. Planejar geração.
6. Executar geração.

## Regras

- Decisões são determinísticas.
- Toda decisão deve ser justificável.
- Toda decisão referencia Evidências.
