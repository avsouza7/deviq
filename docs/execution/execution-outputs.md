# Execution Outputs

## Objetivo

Definir todas as saídas produzidas por uma execução do `deviq run`.

## Saídas Primárias

- Execution Result
- Execution Summary
- Report Model

## Saídas Secundárias

- Knowledge Update Plan
- Artifact Plan
- Quality Report
- Execution Manifest

## Regras

- Toda saída referencia o Run ID.
- Toda saída é rastreável até as Evidências.
- Saídas são produzidas somente após validação do pipeline.
