# AI Runtime Glossary

## Objetivo

Consolidar a terminologia oficial utilizada pela especificação do DevIQ
Runtime.

## Termos

### AI Runtime

Agente de IA que executa o pipeline oficial do DevIQ conforme a
Specification.

### Specification

Conjunto de documentos normativos que definem o comportamento esperado
do Runtime.

### Context Profile

Conjunto de documentos carregados para uma execução específica.

### Context Loading

Processo de carregamento ordenado da documentação oficial.

### Session

Unidade de execução analisada pelo Runtime.

### Interaction

Troca entre usuário e agente durante uma sessão.

### Prompt

Instrução fornecida ao agente.

### Evidence

Informação observável e rastreável extraída da sessão.

### Contribution

Resultado derivado das evidências conforme o Contribution Model.

### Metrics

Indicadores calculados a partir das Contributions.

### Knowledge

Conhecimento consolidado derivado das evidências.

### Reporting

Artefatos de saída produzidos pelo Runtime.

### Golden Dataset

Conjunto de referência utilizado para validação.

### Conformance Suite

Casos de teste oficiais para verificar conformidade.

### Compatibility Matrix

Regras que definem quais versões de artefatos podem ser utilizadas em
conjunto.

### Certification

Processo formal para validar a conformidade de uma implementação.
