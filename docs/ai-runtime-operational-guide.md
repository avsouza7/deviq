# AI Runtime Operational Guide

## Objetivo

Descrever o procedimento operacional para execução do DevIQ Runtime por
um agente de IA.

## Pré-requisitos

-   Documentação oficial do DevIQ disponível.
-   AI Runtime Specification.
-   AI Runtime Prompt.
-   Sessão de entrada.
-   Golden Datasets (quando aplicável).

## Procedimento

1.  Carregar os princípios arquiteturais.
2.  Carregar RFC-001 e RFC-002.
3.  Inicializar o agente com o AI Runtime Prompt.
4.  Processar a sessão seguindo o pipeline oficial.
5.  Validar as saídas com a Conformance Suite.
6.  Registrar Evidence, Contributions, Metrics, Knowledge e Reporting.

## Critérios Operacionais

-   Não inferir evidências inexistentes.
-   Preservar rastreabilidade.
-   Produzir resultados reproduzíveis.
-   Registrar limitações quando houver informação insuficiente.

## Encerramento

A execução é considerada concluída quando todas as saídas obrigatórias
forem produzidas e aprovadas pela Conformance Suite.
