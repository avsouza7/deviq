# Technology Stack

## Objetivo

Definir a stack tecnológica oficial da implementação de referência do
DevIQ.

## Linguagem

-   Java 21 (LTS)

## Build

-   Maven

## Serialização

-   Jackson
-   JSON Schema

## Testes

-   JUnit 5
-   AssertJ

## Qualidade

-   Spotless
-   Checkstyle
-   JaCoCo

## CI

-   GitHub Actions

## Versionamento

-   SemVer
-   Conventional Commits

## Princípios

A stack deve preservar compatibilidade com a arquitetura congelada e
privilegiar portabilidade, determinismo e manutenção de longo prazo.
