# AI Runtime Quickstart

## Objetivo

Permitir que um agente compatível execute o DevIQ Runtime em poucos
passos.

## Passo 1 --- Obter a documentação

Disponibilize ao agente a documentação oficial correspondente à versão
desejada da Specification.

## Passo 2 --- Carregar o contexto

Selecione o Context Profile adequado e carregue a documentação conforme
o AI Runtime Context Loading.

## Passo 3 --- Inicializar o Runtime

Utilize o AI Runtime Prompt para inicializar o agente.

## Passo 4 --- Executar

Forneça a sessão de entrada e execute o pipeline oficial:

1.  Session
2.  Evidence
3.  Contributions
4.  Metrics
5.  Knowledge
6.  Reporting

## Passo 5 --- Validar

Execute a Conformance Suite utilizando os Golden Datasets compatíveis
com a versão da Specification.

## Resultado Esperado

O Runtime produz artefatos compatíveis com os contratos oficiais e
registra a versão da Specification utilizada.
