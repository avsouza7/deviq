# DevIQ

> **Engineering Intelligence for AI-Assisted Development**

DevIQ é um framework de especificação para capturar, estruturar, validar e analisar evidências de colaboração entre desenvolvedores e Inteligência Artificial.

## Objetivo

O projeto define um conjunto de contratos, modelos, schemas, catálogos e métricas para permitir implementações interoperáveis e reproduzíveis, independentemente do provedor de IA utilizado.

## Princípios

- Evidence First
- Contract First
- Provider Agnostic
- Single Responsibility
- Reprodutibilidade
- Determinismo
