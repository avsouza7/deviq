# Confidence Rules

## Objetivo

Definir as regras canônicas para atribuição de confiança às evidências
extraídas durante a análise de uma sessão.

## Princípios

-   Determinismo.
-   Reprodutibilidade.
-   Independência de provider.
-   Base exclusiva nas evidências registradas.

## Regras

1.  Toda evidência deve possuir um nível de confiança explícito ou
    derivável.
2.  O cálculo da confiança deve utilizar apenas informações presentes na
    própria sessão.
3.  Regras de confiança devem produzir o mesmo resultado para a mesma
    entrada.
4.  Valores de confiança não podem alterar a evidência original, apenas
    qualificá-la.

## Resultado

A confiança integra o processo de análise sem modificar o Evidence Model
canônico.
