# Contribution Model

## Objetivo

Definir o modelo canônico de contribuição do DevIQ.

## Conceito

Uma contribuição representa uma unidade de valor identificada durante a
análise de uma sessão e suportada por evidências.

## Estrutura

-   Identificador
-   Tipo
-   Descrição
-   Evidências de suporte
-   Peso
-   Resultado
-   Metadados

## Regras

1.  Toda contribuição deve ser rastreável às evidências de origem.
2.  O peso deve utilizar o catálogo oficial.
3.  O resultado deve utilizar o catálogo oficial.
4.  A mesma entrada deve produzir a mesma contribuição.

## Princípios

-   Evidence First
-   Contract First
-   Determinismo
-   Reprodutibilidade
