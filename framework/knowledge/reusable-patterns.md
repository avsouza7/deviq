# Reusable Patterns

## Objetivo

Catalogar padrões reutilizáveis identificados a partir de evidências
consolidadas.

## Critérios

Um padrão reutilizável deve:

-   possuir evidências de suporte;
-   ser reproduzível;
-   informar contexto de aplicação;
-   registrar limitações conhecidas.

## Estrutura

Cada padrão deve conter:

-   Identificador;
-   Contexto;
-   Problema;
-   Solução;
-   Evidências relacionadas;
-   Métricas observadas;
-   Limitações.

## Princípios

-   Evidence First
-   Determinismo
-   Reprodutibilidade
-   Rastreabilidade
