# Best Prompts

## Objetivo

Catalogar prompts que demonstraram resultados consistentes e
reprodutíveis durante as análises de sessões.

## Critérios

Um prompt pode ser considerado "Best Prompt" quando:

-   apresenta evidências consistentes;
-   produz resultados reproduzíveis;
-   possui rastreabilidade até as evidências de origem;
-   demonstra alinhamento com os princípios do DevIQ.

## Estrutura

Para cada prompt devem ser registrados:

-   Identificador.
-   Objetivo.
-   Contexto de utilização.
-   Evidências associadas.
-   Métricas observadas.
-   Lições aprendidas relacionadas.

## Observações

Este catálogo representa conhecimento derivado e deve ser atualizado
somente a partir de evidências consolidadas.
