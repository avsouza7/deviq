# Session Report Template

## Objetivo

Definir o modelo canônico para apresentação dos resultados de uma
análise de sessão.

## Estrutura

### Identificação

-   Sessão
-   Data da análise
-   Versão do modelo

### Resumo

-   Resultado geral
-   Principais evidências
-   Métricas calculadas

### Evidências

-   Evidências aceitas
-   Evidências rejeitadas (com justificativa)

### Métricas

-   IAI
-   AIPS
-   ICP

### Observações

-   Comentários adicionais quando aplicável.

## Princípios

-   Determinístico
-   Reprodutível
-   Independente de provider
-   Baseado exclusivamente nas evidências registradas
