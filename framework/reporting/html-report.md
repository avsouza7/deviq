# HTML Report

## Objetivo

Definir o formato HTML canônico para apresentação dos resultados da
análise de sessão.

## Estrutura

-   Cabeçalho com identificação da sessão.
-   Resumo executivo.
-   Evidências aceitas e rejeitadas.
-   Métricas (IAI, AIPS e ICP).
-   Observações.

## Regras

1.  O conteúdo deve corresponder ao contrato canônico do relatório.
2.  A estrutura HTML não altera o significado dos dados.
3.  A exportação deve ser determinística.

## Compatibilidade

Compatível com Session Report Template, Markdown Report e JSON Report.
