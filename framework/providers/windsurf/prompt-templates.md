# Prompt Templates

## Objetivo

Definir modelos canônicos de prompts utilizados pelo provider Windsurf
para interagir com modelos de IA de forma consistente.

## Tipos

-   Análise de sessão
-   Extração de evidências
-   Geração de relatório
-   Consolidação de conhecimento

## Regras

1.  Todo template deve ser versionado.
2.  Alterações devem preservar compatibilidade quando possível.
3.  Os templates não podem alterar os contratos oficiais do Framework.
4.  O resultado deve ser convertido para o Evidence Model canônico.

## Princípios

-   Contract First
-   Provider Agnostic
-   Reprodutibilidade
